### Описание
Метод `slice()` возвращает новый массив, копируя указанный диапазон элементов исходного массива. Начало и конец диапазона определяются индексами элементов. Исходный массив при этом не изменяется.

```js
const persons = [  
    { name: 'Egor', age: 24, budget: 40000 },  
    { name: 'Vitaliy', age: 25, budget: 21000 },  
    { name: 'Sasha', age: 34, budget: 18000 },  
    { name: 'Maksim', age: 20, budget: 10000 },  
    { name: 'Andrey', age: 52, budget: 10000 },  
    { name: 'Dima', age: 31, budget: 25000 },  
    { name: 'Artem', age: 18, budget: 17000 },  
]  
  
// Создаст новый массив с элементами основного массива, начиная с индекса 3
const newPersons = persons.slice(3); 
console.log(newPersons);  
//    { name: 'Maksim', age: 20, budget: 10000 },  
//    { name: 'Andrey', age: 52, budget: 10000 },  
//    { name: 'Dima', age: 31, budget: 25000 },  
//    { name: 'Artem', age: 18, budget: 17000 }, 

// Создаст новый массив с тремя последними элементами исходного массива
const last3Persons = persons.slice(-3);  
console.log(last3Persons);
//    { name: 'Andrey', age: 52, budget: 10000 },  
//    { name: 'Dima', age: 31, budget: 25000 },  
//    { name: 'Artem', age: 18, budget: 17000 },  
```

```js
const months = ['янв', 'фев', 'март', 'апр', 'май', 'июнь', 'июль'];  
// Скопирует элементы с индекса 2 по 4 включительно (индекс 5 не входит)
const springMonths = months.slice(2, 5); // ['март', 'апр', 'май']
```

### Параметры
`slice()` принимает два необязательных аргумента:
- _индекс начала_, позиция элемента, начиная с которого будут копироваться элементы;
- _индекс конца_, позиция элемента следующего за последним копируемым элементом в исходном массиве.

>[!Warning]  
>Если индекс отрицательный, то отсчёт элементов ведётся от конца массива.
>
>Если _индекс начала_ больше или равен длине массива, `slice()` вернёт пустой массив.
>
>Если _индекс конца_ не указан или определяется как `end >= array.length`, то концом диапазона копирования будет последний элемент.
>
>Если метод `slice()` вызван без аргументов, будет создана копия всего исходного массива.

### Возвращаемое значение
`Array.slice` возвращает новый массив. ([[Безопасные способы мутировать объект|shallow copy]])
```js
const persons = [  
    { name: 'Egor', age: 24, budget: 40000 },  
    { name: 'Vitaliy', age: 25, budget: 21000 },  
    { name: 'Sasha', age: 34, budget: 18000 },  
    { name: 'Maksim', age: 20, budget: 10000 },  
    { name: 'Andrey', age: 52, budget: 10000 },  
    { name: 'Dima', age: 31, budget: 25000 },  
    { name: 'Artem', age: 18, budget: 17000 },  
]  
  
const newPersons = persons.slice();  
  
newPersons[0].name = 'Ivan';  
  
console.log(persons[0]); // {name: 'Ivan', age: 24, budget: 40000}
```

### Дополнительно
>[!warning] `slice()` с объектами
>Если исходный массив имеет элемент, который является объектом, то возвращаемый в результате работы метода `slice()` массив будет содержать для такого элемента ссылку ([поверхностную копию (shallow copy) элемента](https://doka.guide/js/shallow-or-deep-clone/)), а не новый объект. Таким образом, изменения объекта в исходном массиве будут видны в созданном массиве и наоборот

 Если массив имеет незаполненные элементы, то при выполнении `slice()` массив сохранит для таких элементов отсутствие значения и НЕ изменит их на `undefined`.
