`Псевдомассивы` – это объекты, у которых есть индексы и свойство `length`, то есть, они выглядят как массивы.

>[!note] Не путать с итерируемыми объектами
>При использовании JavaScript в браузере или других окружениях мы можем встретить объекты, которые являются итерируемыми или псевдомассивами, или и тем, и другим.
>
Например, строки итерируемы (для них работает `for..of`) и являются псевдомассивами (они индексированы и есть `length`).
>
Но итерируемый объект может не быть псевдомассивом. И наоборот: псевдомассив может не быть итерируемым.


`Итерируемые объекты` – это объекты, которые реализуют метод [[Функция итератор|Symbol.iterator]].

## Отличие псевдомассива от массива
И итерируемые объекты, и псевдомассивы – это обычно _не массивы_, у них нет методов `push`, `pop` и т.д. Довольно неудобно, если у нас есть такой объект и мы хотим работать с ним как с массивом.

Главным отличием псевдомассива от массива является его наследование прототипа, то есть свойство `__proto__`.  
  
Когда мы посмотрим в свойства массива, то увидим, что он наследует прототип _Array_ объекта. То есть, все свойства, которые есть в объекте _Array.prototype_ будут доступны для любого массива. Если же посмотреть в свойства какого-либо псевдомассива, то можно заметить, что он наследует прототип другого объекта вместе с другими свойствами.

## Как конвертировать псевдомассив в массив?
1. С помощью функции Array.from();
```js
const object = {0: 1, 1: 2, 2: 3, length: 3}// Преобразуем псевдомассив в массив
const array = Array.from(object);
console.log( array ); // [1, 2, 3]
```

2. С помощью функции  `Array.prototype.slice.call()`
Этот метод «наших дедушек и бабушек», который работает до сих пор.
```js
const object = {0: 1, 1: 2, 2: 3, length: 3}// Преобразуем псевдомассив в массив
const array = Array.prototype.slice.call(object); // Или сокращённая форма: 
[].slice.call(object);
console.log( array ); // [1, 2, 3]
```

3. C помощью [[Spread оператор|Spread оператора]]
```js
const object = document.querySelectorAll(selector);// Преобразуем псевдомассив в массив
const array = [...object];
console.log( array ); // [element, element, element]
```
